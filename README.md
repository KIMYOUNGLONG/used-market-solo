# 🛒 중고마켓 솔로 프로젝트 (SpringBoot + PostgreSQL)

> 이 프로젝트는 Spring Boot 기반의 중고거래 웹 플랫폼을 구현한 개인 클론코딩입니다.  
> 회원가입, 로그인, 상품 등록/수정/삭제, 댓글 기능 등을 포함합니다.

---

## ✅ 사용 기술

- Java 17
- Spring Boot, JPA, Thymeleaf
- PostgreSQL
- HTML/CSS
- GitHub + Notion 관리

---

## 🔧 기능 목록

- [x] 회원가입 + 유효성 검사
- [x] 로그인 (세션 기반)
- [x] 상품 등록 / 목록 / 수정 / 삭제
- [ ] 댓글 기능
- [ ] 좋아요 기능
- [ ] 관리자 기능
- [ ] OAuth2 로그인
